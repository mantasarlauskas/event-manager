# Event manager

This project was created to be used along with API. However, API is not available anymore so **data.json** file was created which was filled with some dummy data. All new data is being saved in session storage.

## Roles

- **Guest**
  - Can: 
     - Login
     - Register to events
- **Admin**
  - Can: 
     - View events
     - Edit events
     - Create events
  - Credentials:
     - Email: *root@root*
     - Password: *root*
- **Super Admin**
  - Can: 
     - View events
     - Create admins
  - Credentials:
     - Email: *admin@admin*
     - Password: *admin*
